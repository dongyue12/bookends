## 推送到两个远程服务器
```bash
git remote -v # 查看远程服务器

# 第一个添加的服务器默认是拉取+推送的
git remote add origin <url>   #注意不要箭头<>

# 第二个添加的服务器只能推送,但是此时默认的会被覆盖
git remote set-url --add --push <url>

# 在添加第三个
git remote set-url --add --push <url>

```

## 查看提交历史并回退
```bash
git log --online -5 # 查看最近5次提交
git log --online # 查看所有提交，按q退出

git reset --hard <commit_id> # 回退到指定提交

```

## 本地分支管理
```bash
git branch # 查看本地分支
git branch -M main # 重命名当前分支为main
git branch feature # 创建新功能分支
git checkout <branch_name> # 切换到指定分支
git branch -d <branch_name> # 删除分支

git branch -r # 查看远程分支
```
